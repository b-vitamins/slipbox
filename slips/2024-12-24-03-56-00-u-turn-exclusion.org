#+TITLE: 7.34 U-turn Exclusion
#+FILETAGS: :physics:spof:ch7:series:walker:uturn:
#+SETUPFILE: ~/.config/emacs/setup/setupfile.org

The exclusion of U-turns (immediate backtracking) proves essential for the exact two-dimensional Ising solution, preventing certain cancellations in the signed path sum that would otherwise invalidate the counting procedure. This constraint emerges naturally from the physics while simplifying the mathematical structure.

*Why U-turns Must Be Excluded*:

Consider a path segment with immediate backtracking:
#+begin_example
‚óŹ‚Üí‚Üí‚óŹ‚Üí‚Üí‚óŹ
      ‚Üď
      ‚Üď
‚óŹ‚Üź‚Üź‚óŹ‚Üź‚Üź‚óŹ
  ‚ÜĎ‚ÜĎ
#+end_example

This creates a "double bond" that should cancel with its crossed version [[7.D4]]:
#+begin_example
Same side:        Crossed:
‚óŹ====‚óŹ           ‚óŹ=><==‚óŹ
Weight: +1       Weight: -1
#+end_example

However, a U-turn double bond cannot be created by crossing:
#+begin_example
‚óŹ‚Üí‚Üí‚óŹ            No valid crossed
‚óŹ‚Üź‚Üź‚óŹ            version exists!
#+end_example

Without a canceling partner, U-turn configurations would contribute spuriously to the partition function.

*Implementation in Transfer Matrix*:

The directional notation naturally excludes U-turns:

#+begin_example
Allowed from őľ=1 (‚Üí):    Forbidden:
őľ'=1: ‚Üí‚Üí (continue)      őľ'=3: ‚Üí‚Üź (U-turn)
őľ'=2: ‚Üí‚ÜĎ (left turn)     
őľ'=4: ‚Üí‚Üď (right turn)    
#+end_example

Matrix elements satisfy:
\[\langle\mathbf{r}',\mu'|T^*|\mathbf{r},\mu\rangle = 0 \quad \text{if } \mu' = -\mu\]

This gives $T^*$ its characteristic pattern of zeros preventing immediate reversals.

*Physical Justification*:

In the high-temperature expansion, U-turns would correspond to:
1. Placing a factor $(1 + t\sigma_i\sigma_j)$ twice on the same bond
2. Generating $(1 + t\sigma_i\sigma_j)^2 = 1 + 2t\sigma_i\sigma_j + t^2$
3. The $t^2$ term violates the single-occupancy rule from $(\sigma_i\sigma_j)^2 = 1$

*Mathematical Consequences*:

1. *Reduced connectivity*: Each state connects to only 3 others, not 4
2. *Irreducibility*: No isolated subspaces in transfer matrix
3. *Simplified enumeration*: Fewer paths to count at each length

*Contrast with Phantom Loops*:

Phantom loops [[7.24]] allow all paths including U-turns:
- Transfer matrix $T$ has all non-zero nearest-neighbor elements
- Generates $(2d)^\ell$ walks of length $\ell$
- Over-counts by including unphysical configurations

The exact solution restricts to physical paths:
- Transfer matrix $T^*$ has strategic zeros
- Generates $\sim(2d-1)^\ell$ walks asymptotically
- Shifts critical point from $t_c = 1/(2d)$ to exact value

*Verification*:

The U-turn exclusion can be verified by checking that:
1. All graphs generated by the constrained sum appear in the exact high-T expansion
2. Each graph appears with correct multiplicity
3. The resulting partition function matches other derivation methods

*Generalization*:

While specific to the two-dimensional square lattice Ising model, the principle extends:
- Three dimensions: constrain surface configurations appropriately
- Other lattices: modify forbidden directions based on coordination
- Continuous spins: regulate short-wavelength modes

The U-turn exclusion exemplifies how physical constraints simplify mathematical structures, converting potential complications into elegant restrictions that enable exact solutions.
